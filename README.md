# basic-volume-mixer
Simple and basic volume mixer extension for gnome desktops.




![alt text](https://raw.githubusercontent.com/knohan/assets/refs/heads/main/assets_basic-volume-mixer/example.jpg)



- Some programs (e.g. firefox) requires icons deployed manually to "icons" folder (".svg", ".png"...). Icons can be added also for customisation purposes. The name of the icon file should simply match the name you see when you visit "Settings->Sound->Volume Levels"
